# 圓桌討論 Sub-agent Prompt 範本集

**用途**：這份文件提供八個視角的 Sub-agent 分析 prompt 範本，可在每次圓桌討論時直接使用或調整。
**設計原則**：每個 prompt 讓 agent 能獨立閱讀代碼庫、得出有代碼依據的觀察，不依賴其他 agent 的輸出。

---

## 使用說明

1. 將所有（或部分）prompt 同時發出給多個 agent（平行執行）
2. 等全部完成後，整合為圓桌討論記錄
3. 注意：必須在每次討論前更新「分析背景」區塊中的「上次建議清單」

## 快速使用指南

### 基本場景（4 個視角）

同時發出 Prompt 1-4（Alex、Mia、Sam、Kai），等全部完成後整合為圓桌討論記錄。

### 深度場景（8 個視角）

同時發出全部 8 個 Prompt，整合時分為兩個討論批次：
- **批次 A**（原有四角）：SEO、行銷、部落客、學習者
- **批次 B**（延伸四角）：效能、社群、IA、商業

### 每次使用前更新

每個 Prompt 的「分析背景：上次建議清單」區塊，需在每次討論前更新為上次的建議清單，這樣 agent 才能正確做「實作確認」的工作。
